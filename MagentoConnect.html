Spanisches Sprachpaket für Magento Commerce

(Magento Community Modules - Spanish (Colombia) Language Pack)

(Paquete de idioma español para Magento Commerce)

Aviso / Hint / Hinweis

Magento viene en su configuración básica con Español de Castilla. Cualquier persona que desee utilizar Español columbiano, debe instalar el paquete de idioma LocalePack_es_CO.

Magento comes in its basic configuration with Spanish from Spain. Anyone wishing to use Columbian Spanish, should install the language pack LocalePack_es_CO.

Magento kommt in der Grundausstattung mit Spanisch aus Spanien daher. Wer kolumbianisches Spanisch nutzen möchte, sollte das Sprachpaket LocalePack_es_CO installieren.
